# _01) Tutorials

This directory contains brief, focused tutorials demonstrating essential .NET MAUI concepts:
- Setting up the project structure for multi-target platforms (Android, iOS/macOS, Windows).
- Building and running a simple MAUI app with cross-platform UI.
- Basic navigation, styling, and resource usage.
